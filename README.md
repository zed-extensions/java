# Zed Java

This extension adds support for the [Java](https://github.com/zed-extensions/java) language.

## Configuration

You can optionally configure the Java home that JDTLS (the language server) uses like so:

```json
{
  "lsp": {
    "jdtls": {
      "settings": {
        "java_home": "/path/to/jdk"
      }
    }
  }
}
```
